Authentication Flow

Inbound Authentication

When a user connects to the MCP server, Descope's inbound app validates their session token and provides user context.

Outbound Authentication

When tools need to access external services, Descope's outbound apps manage the OAuth tokens, handling refresh and rotation automatically.

Troubleshooting

If you encounter authentication issues, try clearing the authentication files: 

rm -rf ~/.mcp-auth
